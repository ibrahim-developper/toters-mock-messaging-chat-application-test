# Getting Started
This project uses the Gradle build system. To build this project, use "Import Project" in Android Studio or download as a zip file and extract and open in android studio.
* In this project:
    * minSdkVersion 18
    * targetSdkVersion 30
    * Android Studio Version 4.0.1
